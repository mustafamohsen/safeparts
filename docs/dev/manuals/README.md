# Developer manuals

These manuals cover integration and automation topics that need more detail than the surface guides.

- [Rust library integration](rust-library.md): use `safeparts_core` from another Rust project, including API reference and error handling.
- [CLI automation](cli-automation.md): script `safeparts` safely in local jobs and CI pipelines.

Use the surface guides when you are changing Safeparts itself. Use these manuals when you are embedding or automating it.
