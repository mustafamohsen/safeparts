# WASM bindings

Owner: `crates/safeparts_wasm/`
Nearest contract: [`crates/safeparts_wasm/AGENTS.md`](../../../crates/safeparts_wasm/AGENTS.md)

## What belongs here

The WASM crate exposes a small wasm-bindgen API for the web app.

It should:

- delegate split/combine, packet parsing, and encoding behavior to `safeparts_core`
- return browser-friendly values
- keep error messages useful without including share text or secrets
- preserve the API shape consumed by `web/src/wasm.ts`

## Generated package

The browser package is generated by:

```bash
cd web
bun run build:wasm
```

Output goes to `web/src/wasm_pkg/` and is ignored by git.

## Useful checks

```bash
cargo test -p safeparts_wasm
cd web && bun run build:wasm
cd web && bun run typecheck
```

## When WASM changes

Update:

- `web/src/wasm.ts` types or adapter logic
- web tests when UI behavior changes
- [`docs/dev/feature-matrix.md`](../feature-matrix.md)
