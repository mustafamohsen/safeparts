# Generated artifacts

Generated files should be easy to refresh and easy to review. Do not hand-edit generated output unless a task explicitly asks for it.

## Policy

| Path | Generated by | Tracked? | Rule |
| --- | --- | --- | --- |
| `target/` | Cargo | No | Build output. Never edit or commit. |
| `dist/` | release scripts | No | Local release output. Never edit or commit. |
| `web/node_modules/` | Bun | No | Dependency install output. Never commit. |
| `web/dist/` | Vite and docs build | No | Static build output. Never commit. |
| `web/src/wasm_pkg/` | `web/scripts/build-wasm.sh` | No | Regenerate with `bun run build:wasm`. Do not hand-edit. |
| `web/help/node_modules/` | Bun | No | Dependency install output. Never commit. |
| `web/help/dist/` and `web/help/.astro/` | Astro | No | Build/cache output. Never commit. |
| `desktop/node_modules/` | Bun | No | Dependency install output. Never commit. |
| `desktop/dist/` | Vite | No | Desktop frontend build output. Never commit. |
| `desktop/src-tauri/gen/schemas/` | Tauri | Partially tracked today | Treat as generated. Refresh intentionally and review schema diffs. |
| `macos/Generated/` | `macos/scripts/prepare.sh` / UniFFI | Yes | Canonical generated Swift, header, and module maps. Review and commit them together. |
| `macos/Sources/SafepartsKit/Generated.swift` | `macos/scripts/prepare.sh` / UniFFI | Yes | Compiled mirror of `macos/Generated/safeparts_swift.swift`; the prepare script verifies an exact match. |
| `macos/Native/` and `macos/.build/` | Cargo and SwiftPM | No | Compiled host output. Never commit. |
| `mobile/node_modules/` | mobile tooling | No | Local dependency output. Never commit. |
| `mobile/src-native/target/` | Cargo/mobile tooling | No | Build output. Never commit. |

## Refresh commands

WASM package:

```bash
cd web
bun run build:wasm
```

Web build output:

```bash
cd web
bun run build
```

Help docs build output:

```bash
cd web/help
bun run build
```

Desktop frontend build output:

```bash
cd desktop
bun run build
```

Native macOS bindings:

```bash
mise run macos:prepare
cmp macos/Generated/safeparts_swift.swift macos/Sources/SafepartsKit/Generated.swift
```

The preparation script targets macOS 14 by default and strips trailing whitespace from generated text. You may set `MACOSX_DEPLOYMENT_TARGET` to a newer version for diagnostics. The script rejects versions older than 14.0.

Tauri schemas are generated by the Tauri toolchain. If a Tauri upgrade or permission change updates `desktop/src-tauri/gen/schemas/`, review the diff and either commit the intentional schema update or remove local generated drift.

## Lockfiles

Bun is the JS package manager for web, docs, and desktop.

Tracked lockfiles:

- `web/bun.lock`
- `web/help/bun.lock`
- `desktop/bun.lock`

Do not add npm, pnpm, or yarn lockfiles for these packages unless the project deliberately changes package managers.

## Review checklist

Before committing:

```bash
git status --short
mise run dx:verify
```

Do not commit generated output unless this page says the path is tracked or the task explicitly requires it.
