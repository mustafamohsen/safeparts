## Short Description / Subtitle

- Offline-first threshold secret sharing

## Description

Safeparts splits a secret into multiple shares (k-of-n) so you can store them safely across people or places. Recovery works locally on-device and does not require an account or a network connection.

Key features:

- Split and combine shares with multiple encodings
- Optional passphrase protection
- QR export + QR scan import for shares
- Offline help/docs
- English + Arabic with RTL/LTR support

## Privacy

- No accounts
- No analytics/telemetry
- No network transmission of secrets/shares/passphrases
