import { LinkButton, Steps } from '@astrojs/starlight/components';

The web UI lives at the site root (<span class="token token--mono">/</span>).
Split/combine runs locally in your browser via WASM.

<LinkButton href="/" icon="external" target="_blank" rel="noopener noreferrer">
	Open the web app
</LinkButton>

## Split

<Steps>
1. Paste the secret into **Secret**.
2. Choose <var>k</var> and <var>n</var>.
3. Choose an encoding.
4. (Optional) set a passphrase.
5. Click **Split** and store each share separately.
</Steps>

## Combine

<Steps>
1. Set the encoding (or paste a share and let the UI auto-detect).
2. Paste at least <var>k</var> shares.
3. Enter the passphrase if you used one.
4. Click **Combine**.
</Steps>

## Privacy boundary

The app does not upload your secret by default.
Anything you copy/paste or store is your responsibility.
