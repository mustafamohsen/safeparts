اسم binary الخاص بواجهة TUI هو <span class="token token--mono" dir="ltr">safeparts-tui</span>. يمكنك تشغيلها أيضاً عبر CLI باستخدام <span class="token token--mono" dir="ltr">safeparts tui</span>.

استخدم TUI عندما تريد سير عمل موجهاً داخل الطرفية، وتشغيلاً دون اتصال، ودعم clipboard، وإجراءات تحميل أو حفظ للملفات.

## ماذا تدعم TUI

- سير التقسيم والاسترداد.
- كل الترميزات الملموسة: <span class="token" dir="ltr">base64url</span> و<span class="token" dir="ltr">base58check</span> و<span class="token" dir="ltr">mnemo-words</span> و<span class="token" dir="ltr">mnemo-bip39</span>.
- كشف تلقائي عند الاسترداد.
- عبارات مرور اختيارية.
- تحميل ملفات الأسرار أو ملفات الحصص.
- تصدير ملف لكل حصة أو حفظ البايتات المستعادة.

## سير عمل دون اتصال

<Steps>
1. استخدم جهازاً مفصولاً عن الشبكة إذا كان الإجراء يتطلب ذلك.
2. شغّل <span class="token token--mono" dir="ltr">safeparts-tui</span>.
3. اختر <var>k</var> و<var>n</var> والترميز.
4. أدخل السر أو حمّله.
5. قسّم وصدّر الحصص إلى وسائط أو حاملين منفصلين.
6. استرد مرة واحدة باستخدام <var>k</var> حصص إذا سمحت السياسة بالتحقق.
</Steps>

:::caution
التشغيل دون اتصال لا يفيد إلا إذا كان التخزين منفصلاً. لا تضع عدداً كافياً من الحصص للاسترداد على USB واحد أو حساب واحد أو خزنة واحدة.
:::

## اختصارات لوحة المفاتيح

| المفاتيح | الإجراء |
| --- | --- |
| <kbd>Tab</kbd> / <kbd>Shift</kbd>+<kbd>Tab</kbd> | نقل التركيز. |
| <kbd>Left</kbd> / <kbd>Right</kbd> | التبديل بين تبويبي التقسيم والاسترداد. |
| <kbd>Up</kbd> / <kbd>Down</kbd> | تغيير القيم الرقمية أو الترميز المحدد أو الحصة المحددة حسب موضع التركيز. |
| <kbd>Enter</kbd> | تنفيذ التقسيم أو الاسترداد. |
| <kbd>Ctrl</kbd>+<kbd>L</kbd> | تحميل ملف سر أو ملفات حصص. |
| <kbd>Ctrl</kbd>+<kbd>S</kbd> | الحفظ أو التصدير. |
| <kbd>Ctrl</kbd>+<kbd>C</kbd> | نسخ الحصة المحددة أو كل الحصص أو الناتج المستعاد حسب موضع التركيز. |
| <kbd>Ctrl</kbd>+<kbd>V</kbd> | اللصق في المحرر المركز عندما يتوفر clipboard. |
| <kbd>Ctrl</kbd>+<kbd>U</kbd> | مسح حقل عبارة المرور المركز. |
| <kbd>?</kbd> أو <kbd>F1</kbd> | فتح المساعدة. |
| <kbd>Esc</kbd> | إغلاق المساعدة أو إلغاء نافذة. |
| <kbd>Ctrl</kbd>+<kbd>Q</kbd> | الخروج. |
