ملف التنفيذ هو <span class="token token--mono" dir="ltr">safeparts-tui</span>.
يمكنك أيضاً تشغيله عبر CLI: <span class="token token--mono" dir="ltr">safeparts tui</span>.

## متى تستخدم TUI

- عندما تريد العمل offline (جهاز معزول).
- عندما تريد سير عمل تفاعلياً موجهاً.
- عندما تريد حفظ ملف لكل حصة.

## سير عمل offline

<Steps>
1. استخدم جهازاً offline (بدون شبكة).
2. شغّل <span class="token token--mono" dir="ltr">safeparts-tui</span>.
3. اضبط <var>k</var> و <var>n</var> والترميز.
4. قسّم ثم احفظ كل حصة على وسيط منفصل.
5. استعد مرة واحدة باستخدام <var>k</var> حصص للتحقق من الخطة.
</Steps>

:::caution
العمل offline يفيد فقط إذا كانت الحصص مخزنة بشكل منفصل.
لا تضع حصتين على نفس USB.
:::

## اختصارات لوحة المفاتيح

- <kbd dir="ltr">Enter</kbd>: تنفيذ التقسيم/الاسترداد
- <kbd dir="ltr">Ctrl</kbd>+<kbd dir="ltr">L</kbd>: تحميل ملفات السر/الحصص
- <kbd dir="ltr">Ctrl</kbd>+<kbd dir="ltr">S</kbd>: حفظ (ملف لكل حصة أو السر المستعاد)
- <kbd dir="ltr">Ctrl</kbd>+<kbd dir="ltr">C</kbd>: نسخ
- <kbd dir="ltr">Ctrl</kbd>+<kbd dir="ltr">Q</kbd>: خروج
- <kbd dir="ltr">?</kbd>: نافذة مساعدة
