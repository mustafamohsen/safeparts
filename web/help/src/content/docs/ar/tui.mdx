---
title: TUI
description: واجهة طرفية تفاعلية لتقسيم الأسرار واستعادتها بشكل عملي.
---

ملف التنفيذ هو `safeparts-tui` (الحزمة: `crates/safeparts_tui`).

تشغيل مباشر:

```bash
cargo run -q -p safeparts_tui
```

أو عبر مشغّل CLI:

```bash
cargo run -q -p safeparts -- tui
```

## اختصارات لوحة المفاتيح

- <kbd dir="ltr">Enter</kbd>: تنفيذ التقسيم/الاستعادة
- <kbd dir="ltr">Ctrl</kbd>+<kbd dir="ltr">L</kbd>: تحميل ملفات السر/الحصص
- <kbd dir="ltr">Ctrl</kbd>+<kbd dir="ltr">S</kbd>: حفظ (ملف لكل حصة أو السر المستعاد)
- <kbd dir="ltr">Ctrl</kbd>+<kbd dir="ltr">C</kbd>: نسخ (UTF-8 إن أمكن، وإلا <span class="token" dir="ltr">base64</span>)
- <kbd dir="ltr">Ctrl</kbd>+<kbd dir="ltr">Q</kbd>: خروج
- <kbd dir="ltr">?</kbd>: نافذة مساعدة

:::note
دعم الحافظة يستخدم حافظة النظام عند توفرها، ويعود إلى تسلسلات OSC52 عند الحاجة
(مفيد عبر SSH/tmux عندما يدعمها الطرفية).
:::
