---
title: TUI
description: واجهة طرفية تفاعلية لتقسيم الأسرار واستعادتها بشكل عملي.
---

ملف التنفيذ هو `safeparts-tui` (الحزمة: `crates/safeparts_tui`).

تشغيل مباشر:

```bash
cargo run -q -p safeparts_tui
```

أو عبر مشغّل CLI:

```bash
cargo run -q -p safeparts -- tui
```

## اختصارات لوحة المفاتيح

- `Enter`: تنفيذ التقسيم/الاستعادة
- `Ctrl+L`: تحميل ملفات السر/الحصص
- `Ctrl+S`: حفظ (ملف لكل حصة أو السر المستعاد)
- `Ctrl+C`: نسخ (UTF-8 إن أمكن، وإلا base64)
- `Ctrl+Q`: خروج
- `?`: نافذة مساعدة

:::note
دعم الحافظة يستخدم حافظة النظام عند توفرها، ويعود إلى تسلسلات OSC52 عند الحاجة
(مفيد عبر SSH/tmux عندما يدعمها الطرفية).
:::
