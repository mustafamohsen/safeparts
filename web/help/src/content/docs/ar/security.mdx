Safeparts مصمم بحيث تبقى *مادة السر* على جهازك أثناء إنشاء حصص الاستعادة.

## خصائص تطبيق الويب

- بدون Cookies.
- بدون تحليلات/تتبع.
- التقسيم والاستعادة تتم محلياً (WASM).
- مفتوح المصدر وقابل للاستضافة الذاتية.

:::note
قد تحفظ Starlight (هذا الموقع) وواجهة الويب تفضيلات غير حساسة داخل `localStorage`
(مثل المظهر). هذا ليس Cookie.
:::

## ماذا يعني «محلياً» هنا؟

Safeparts لا يرسل سرك إلى خادم لمعالجته.

ومع ذلك، قد تغادر البيانات جهازك عندما تختار ذلك:

- نسخ/لصق الحصص في تطبيقات أخرى.
- حفظ الحصص في مجلدات تتم مزامنتها.
- مشاركة الشاشة أو اللصق في الدردشة.

:::caution
تعامل مع الحصص كأنها السر نفسه. إذا حصل مهاجم على <var>k</var> حصص، يمكنه الاستعادة.
:::

## السلامة وعبارة المرور

- يضيف Safeparts وسم سلامة (BLAKE3) كي تفشل الاستعادة عند وجود حصص تالفة/غير متطابقة.
- عند استخدام عبارة مرور، يقوم Safeparts بالتشفير قبل التقسيم (Argon2id -> ChaCha20-Poly1305).

## الاستضافة الذاتية

لاستضافة الموقع كاملاً (التطبيق + الوثائق) كحزمة Static واحدة:

```bash
cd web
bun install
bun run build:wasm
bun run build
bun run help:build
```

انشر مجلد <span class="token token--mono" dir="ltr">web/dist/</span>.
