title: الترميزات والحزم

يعيد التقسيم قيماً ثنائية من نوع `SharePacket`. يجب على معظم التطبيقات ترميز الحزم قبل التخزين أو العرض أو النقل.

## ترميز الحزم

استخدم `safeparts_core::encoding` كواجهة الترميز النصي العامة.

```rust
use safeparts_core::encoding::{self, Encoding};
use safeparts_core::{split_secret, CoreResult};

fn encode_for_storage() -> CoreResult<Vec<String>> {
    let packets = split_secret(b"synthetic storage secret", 2, 3, None)?;

    packets
        .iter()
        .map(|packet| encoding::encode_packet(packet, Encoding::Base64url))
        .collect()
}
```

`Encoding::Auto` للتحليل فقط. يجب أن يختار الخرج ترميزاً ملموساً واحداً.

## تحليل مدخلات ملصقة

```rust
use safeparts_core::encoding::{self, Encoding};
use safeparts_core::{combine_shares, CoreResult};

fn recover_from_paste(input: &str) -> CoreResult<Vec<u8>> {
    let parsed = encoding::parse_share_packets(input, Encoding::Auto)?;
    combine_shares(&parsed.packets, None)
}
```

في واجهات النص حيث قد تلتف حصة mnemonic طويلة على عدة أسطر، استخدم `parse_share_packets_wrapped_mnemonics`. تظل الأسطر الفارغة فاصلاً بين حصص mnemonic.

## الترميزات المدعومة

| الترميز | الاستخدام |
| --- | --- |
| `Encoding::Base64url` | نص مضغوط وآمن للروابط. |
| `Encoding::Base58check` | نص مضغوط مع checksum من Base58Check. |
| `Encoding::MnemoWords` | مجموعات كلمات مع CRC16 للإدخال اليدوي. |
| `Encoding::MnemoBip39` | تسلسلات كلمات BIP-39 صالحة لمسارات التخزين المتوافقة. |
| `Encoding::Auto` | كشف parse-only لمدخلات المشغلين. |

## بيانات الحزمة

تحتوي كل `SharePacket` على:

- `set_id`: معرّف عشوائي لمجموعة التقسيم؛
- `k` و`n`: العتبة وإجمالي عدد الحصص؛
- `x`: إحداثي الحصة؛
- `payload`: بايتات الحصة؛
- `crypto_params`: معلمات Argon2id وnonce الاختيارية للتقسيمات المشفرة.

هذه البيانات ليست سرية. تساعد combine على رفض sets مختلطة، وفهارس مكررة، ومعلمات تشفير غير متوافقة.

## نمط تخزين

تدفق شائع في التطبيقات:

1. قسّم البايتات إلى قيم `SharePacket`.
2. رمّز كل حزمة بترميز ملموس واحد.
3. خزّن كل حصة مرمزة في نطاق فشل منفصل.
4. خزّن اسم الترميز مع metadata الحيازة إذا لن يستخدم المشغلون الكشف التلقائي.
5. لاحقاً حلل على الأقل <var>k</var> حصص مرمزة.
6. ادمج الحزم وتعامل مع البايتات المستعادة كناتج حساس.

لا تحتفظ بعدد كافٍ من الحصص معاً في صف قاعدة بيانات واحد أو عنصر vault واحد أو CI secret store واحد أو stream سجلات واحد إلا إذا كانت السياسة تعامل ذلك الموقع مثل السر الأصلي.
