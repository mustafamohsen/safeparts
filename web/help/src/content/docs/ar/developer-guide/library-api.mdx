استخدم `safeparts_core` عندما يحتاج تطبيق Rust إلى سلوك Safeparts دون استدعاء CLI.

تعمل crate على البايتات. تتعامل مع التقسيم والاسترداد والتحقق من الحزم وفحوص السلامة والترميزات النصية وحماية عبارة المرور الاختيارية. يبقى على تطبيقك التخزين، والتفويض، وسياسة التدقيق، وعمر الأسرار.

:::caution
لا تسجّل الأسرار أو البايتات المستعادة أو عبارات المرور أو الحصص المرمزة أو payload الحزم.
:::

## إضافة crate

من repository آخر، استخدم Git dependency إلى أن تتوفر crate منشورة:

```toml
[dependencies]
safeparts_core = { git = "https://github.com/mustafamohsen/safeparts", package = "safeparts_core" }
```

داخل هذا workspace، اعتمد على crate المحلية حسب بنية crate عندك.

إذا كان تطبيقك يلف buffers حساسة، أضف `zeroize`:

```toml
[dependencies]
zeroize = "1"
```

## تقسيم واسترداد البايتات

```rust
use safeparts_core::{combine_shares, split_secret, CoreResult};

fn split_then_recover() -> CoreResult<()> {
    let secret = b"synthetic integration secret";

    let shares = split_secret(secret, 2, 3, None)?;
    let recovered = combine_shares(&shares[..2], None)?;

    assert_eq!(recovered, secret);
    Ok(())
}
```

مجال العتبة في النواة هو <span class="token" dir="ltr">1 &lt;= k &lt;= n &lt;= 255</span>. يحتاج الاسترداد إلى <var>k</var> حزم على الأقل من مجموعة التقسيم نفسها.

## ترميز الحصص للتخزين

```rust
use safeparts_core::encoding::{self, Encoding};
use safeparts_core::{split_secret, CoreResult};

fn split_for_storage() -> CoreResult<Vec<String>> {
    let packets = split_secret(b"synthetic storage secret", 2, 3, None)?;

    packets
        .iter()
        .map(|packet| encoding::encode_packet(packet, Encoding::Base64url))
        .collect()
}
```

خزّن كل حصة مرمزة في نطاق فشل منفصل. إبقاء عدد كافٍ من الحصص معاً يهدم خطة العتبة.

## تحليل الحصص الملصقة والاسترداد

```rust
use safeparts_core::encoding::{self, Encoding};
use safeparts_core::{combine_shares, CoreResult};

fn recover_from_paste(input: &str) -> CoreResult<Vec<u8>> {
    let parsed = encoding::parse_share_packets(input, Encoding::Auto)?;
    combine_shares(&parsed.packets, None)
}
```

استخدم `parse_share_packets_wrapped_mnemonics` لمناطق النص حيث قد تلتف حصة mnemonic واحدة على عدة أسطر. تفصل الأسطر الفارغة بين حصص mnemonic الملتفة.

## إضافة حماية عبارة مرور

```rust
use safeparts_core::{combine_shares, split_secret, CoreResult};
use zeroize::Zeroizing;

fn encrypted_round_trip() -> CoreResult<()> {
    let passphrase = Zeroizing::new(b"synthetic passphrase for tests".to_vec());

    let shares = split_secret(
        b"synthetic protected secret",
        2,
        3,
        Some(passphrase.as_slice()),
    )?;

    let recovered = combine_shares(&shares[..2], Some(passphrase.as_slice()))?;
    assert_eq!(recovered, b"synthetic protected secret");
    Ok(())
}
```

تخزّن الحزم الملح وnonce ومعلمات Argon2id المطلوبة للاسترداد. لا تخزّن عبارة المرور.

## مرجع API العام

### واجهة crate العليا

| العنصر | الاستخدام |
| --- | --- |
| `split_secret(secret, k, n, passphrase)` | API الرئيسي لإنشاء حزم الحصص. |
| `combine_shares(packets, passphrase)` | API الرئيسي لاسترداد بايتات السر. |
| `tag_and_split(secret, k, n)` | غلاف توافق للتقسيم دون حماية. فضّل `split_secret` في الكود الجديد. |
| `combine_and_verify(packets)` | غلاف توافق للاسترداد دون حماية. فضّل `combine_shares` في الكود الجديد. |
| `CoreError` | enum للأخطاء typed. |
| `CoreResult<T>` | alias لنتائج عمليات النواة. |

### وحدة `encoding`

| العنصر | الاستخدام |
| --- | --- |
| `Encoding` | `Auto`, `Base64url`, `Base58check`, `MnemoWords`, و`MnemoBip39`. |
| `Encoding::CONCRETE` | ترميزات الخرج الملموسة. |
| `Encoding::WITH_AUTO` | الترميزات الملموسة مع التحليل التلقائي. |
| `Encoding::label()` | تسمية canonical للواجهة أو metadata التخزين. |
| `Encoding::parse_name(name)` | يحلل الأسماء canonical وأسماء CLI المستعارة. |
| `ParsedSharePackets` | الحزم المحللة مع الترميز المكتشف. |
| `encode_packet(packet, encoding)` | ترميز حزمة واحدة كنص. |
| `decode_packet(text, encoding)` | تحليل حزمة واحدة. |
| `parse_share_packets(input, encoding)` | تحليل حصة أو أكثر. |
| `parse_share_packets_wrapped_mnemonics(input, encoding)` | تحليل مدخلات منطقة نص مع حصص mnemonic ملتفة. |
| `detect_encoding(input)` | كشف الترميز المحتمل دون إرجاع الحزم. |

### وحدة `packet`

| العنصر | الاستخدام |
| --- | --- |
| `SharePacket` | حزمة فيها `set_id` و`k` و`n` و`x` وpayload ومعلمات تشفير اختيارية. |
| `SharePacket::is_encrypted()` | يرجع true عند استخدام حماية عبارة مرور. |
| `SharePacket::encode_binary()` | serializes إلى صيغة حزمة Safeparts الثنائية. |
| `SharePacket::decode_binary(bytes)` | يحلل صيغة الحزمة الثنائية. |
| `packet::binary_total_len(bytes)` | يحسب طول الحزمة الكامل من بادئة بايتات. |

## مسؤوليات التطبيق

لا يقرر `safeparts_core`:

- أين تُخزن كل حصة؛
- من يحق له تقديم حصة؛
- هل عبارات المرور مطلوبة؛
- كيف تتحرك البايتات المستعادة في الذاكرة أو على القرص؛
- أي أحداث تدقيق آمنة للتسجيل.

حدث التدقيق الآمن يقول إن الاسترداد بدأ أو اكتمل تحت سياسة معتمدة. يجب ألا يتضمن نص الحصص أو القيم المستعادة.
