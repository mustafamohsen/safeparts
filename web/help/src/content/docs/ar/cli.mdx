اسم binary الخاص بواجهة CLI هو <span class="token token--mono" dir="ltr">safeparts</span>. يقرأ من stdin افتراضياً ويكتب إلى stdout إلا إذا مررت مسارات ملفات.

استخدم CLI عندما تحتاج إلى أوامر قابلة للتكرار أو مدخلات ثنائية أو كل الترميزات المدعومة أو runbooks للمشغلين.

## التقسيم

قسّم stdin إلى ثلاث حصص، مع طلب أي حصتين للاسترداد:

```bash
printf '%s' 'synthetic secret' | safeparts split -k 2 -n 3 -e base64url
```

قسّم ملفاً واكتب الحصص إلى ملف:

```bash
safeparts split \
  --threshold 2 \
  --shares 3 \
  --encoding mnemo-words \
  --in secret.bin \
  --out shares.txt
```

يحتوي ملف الخرج على حصة واحدة في كل سطر للترميزات المضغوطة. لا تحتفظ بملف يحتوي كل الحصص إلا إذا كانت سياستك تتعامل معه مثل السر الأصلي.

## الاسترداد

استرد من stdin:

```bash
printf '%s\n%s\n' '<share1>' '<share2>' | safeparts combine
```

استرد من ملف واكتب البايتات المستعادة إلى ملف:

```bash
safeparts combine --in selected-shares.txt --out recovered-secret.bin
```

إذا لم يستطع الكشف التلقائي تحديد الصيغة، مررها صراحة:

```bash
safeparts combine --encoding base64url --in selected-shares.txt --out recovered-secret.bin
```

## الترميزات

يدعم `split` هذه الترميزات في الخرج:

- <span class="token" dir="ltr">base64url</span>، مع الاسم المستعار <span class="token" dir="ltr">base64</span>
- <span class="token" dir="ltr">base58check</span>، مع الاسم المستعار <span class="token" dir="ltr">base58</span>
- <span class="token" dir="ltr">mnemo-words</span>
- <span class="token" dir="ltr">mnemo-bip39</span>

يمكن لـ `combine` كشف الترميز تلقائياً عند حذف `--encoding`.

## عبارات المرور

فضّل <span class="token token--mono" dir="ltr">--passphrase-file</span> أو <span class="token token--mono" dir="ltr">-P</span>. قد تظهر عبارات المرور المكتوبة كوسيط أمر في تاريخ shell وقوائم العمليات ومسجلات الطرفية وأدوات التدقيق.

```bash
safeparts split -k 2 -n 3 -e base64url -P passphrase.txt -i secret.bin -o shares.txt
safeparts combine -P passphrase.txt -i selected-shares.txt -o recovered-secret.bin
```

## شكل runbook آمن

<Steps>
1. اختبر تسلسل الأوامر بسر اصطناعي.
2. قسّم السر الحقيقي من stdin أو ملف متحكم به.
3. تحقق من الاسترداد باستخدام <var>k</var> حصص بالضبط إذا سمحت السياسة بذلك.
4. وزّع الحصص على حاملين أو مواقع منفصلة.
5. أزل الملفات المؤقتة التي تحتوي نصاً صريحاً أو ناتجاً مستعاداً أو كل الحصص معاً.
6. سجّل الحيازة والعتبة والترميز وخطوات الموافقة دون تسجيل نص الحصص.
</Steps>
