---
title: الترميزات
description: اختر صيغة نصية للحصة تناسب التخزين أو النسخ واللصق أو الإدخال اليدوي.
---

الترميز يغيّر طريقة كتابة حزمة الحصة فقط. لا يغيّر عدد الحصص المطلوبة أو رياضيات المشاركة أو حماية عبارة المرور.

اختر الصيغة التي يستطيع سير الاسترداد تخزينها وإعادة إدخالها بدقة.

## الصيغ

| الترميز | أفضل استخدام | ملاحظات |
| --- | --- | --- |
| <span class="token" dir="ltr">base64url</span> | مدراء كلمات المرور والملفات والـ scripts والـ APIs | نص مضغوط وآمن للروابط. اسم مستعار في CLI: <span class="token" dir="ltr">base64</span>. |
| <span class="token" dir="ltr">base58check</span> | نسخ ولصق يدوي مع نص مضغوط | يتجنب بعض الأحرف المتشابهة بصرياً ويضيف checksum من Base58Check. اسم مستعار في CLI: <span class="token" dir="ltr">base58</span>. |
| <span class="token" dir="ltr">mnemo-words</span> | الورق والمعدن والاسترداد المنطوق | يستخدم مجموعات كلمات مع CRC16 لاكتشاف كثير من أخطاء النسخ. |
| <span class="token" dir="ltr">mnemo-bip39</span> | مسارات تخزين تتوقع تسلسلات كلمات BIP-39 صالحة | قد يبدو نص الحصة مثل كلمات محفظة، لكنه ليس seed لمحفظة. |

## التوفر حسب الواجهة

| الواجهة | ناتج التقسيم | مدخلات الاسترداد |
| --- | --- | --- |
| تطبيق الويب | <span class="token" dir="ltr">base64url</span> و<span class="token" dir="ltr">mnemo-words</span> | الصيغتان نفسيهما، مع كشف best-effort عند اللصق. |
| تطبيق Tauri لسطح المكتب | <span class="token" dir="ltr">base64url</span> و<span class="token" dir="ltr">mnemo-words</span> | الصيغتان نفسيهما، عبر أوامر Tauri محلية. |
| تطبيق macOS الأصلي | كل الترميزات الأربعة الملموسة | يكتشف metadata تلقائياً أو يقبل صيغة محددة. |
| CLI | كل الترميزات الأربعة الملموسة | كشف تلقائي عند حذف `--encoding`، أو اختيار ترميز محدد. |
| TUI | كل الترميزات الأربعة الملموسة | كشف تلقائي أو اختيار ترميز محدد. |
| مكتبة Rust | كل الترميزات الأربعة عبر `safeparts_core::encoding` | APIs للكشف التلقائي أو التحليل المحدد. |

## اختيار صيغة

استخدم <span class="token" dir="ltr">base64url</span> عندما تخزّن أو تنقل الآلات الحصص. هو الخيار الشائع الأقصر ويعمل جيداً في الملفات وحقول vault.

استخدم <span class="token" dir="ltr">base58check</span> عندما يقرأ المشغلون نصاً مضغوطاً بالعين وتريد checksum على مستوى الترميز.

استخدم <span class="token" dir="ltr">mnemo-words</span> عندما سينسخ البشر الحصص أو يطبعونها أو يقرؤونها أو ينقشونها.

استخدم <span class="token" dir="ltr">mnemo-bip39</span> فقط عندما تساعد تسلسلات الكلمات الصالحة لـ BIP-39 في مسار التخزين.

حصص Safeparts بصيغة mnemonic ليست عبارات seed لمحافظ العملات. لا تستوردها في برامج المحافظ.
