## واجهة الويب تعرض `WASM module not found`

ابنِ حزمة WASM ثم أعد التحميل:

```bash
cd web
bun run build:wasm
bun run dev
```

على Windows راجع قسم Windows في [البناء والتشغيل](../build-and-run/).

## أمر combine لا يستطيع اكتشاف الصيغة

إذا فشل الكشف التلقائي، حدّدها صراحة:

```bash
safeparts combine --encoding mnemo-words
```

## فشل التحقق من السلامة أثناء الاستعادة

غالباً يعني ذلك أحد الأمور التالية:

- هناك حصة واحدة على الأقل مكتوبة بشكل خاطئ/تالفة.
- الحصص ليست من نفس المجموعة (اختلاف metadata).
- عبارة المرور خاطئة (إذا كان التشفير مستخدماً).

## الحافظة لا تعمل في TUI

بعض البيئات تمنع الوصول إلى الحافظة. سيحاول `safeparts-tui` استخدام حافظة النظام، ثم
يعود إلى OSC52 عند الإمكان. إذا لم يعمل ذلك، استخدم اختصار الحفظ إلى ملف ثم انسخ من
المحرر/الطرفية.
