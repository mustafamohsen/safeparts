واجهة CLI غير تفاعلية ما لم يضف runbook مطالبات حولها. هذا يجعلها موثوقة في الإجراءات، لكنه يعني أيضاً أن تاريخ shell وscrollback الطرفية والملفات المؤقتة مهمة.

## التقسيم من ملف متحكم به

استخدم ملف إدخال للاعتمادات الحقيقية عندما يكون ذلك ممكناً. أنشئ الملف في مجلد عمل خاص وأزله بعد التحقق.

```bash
safeparts split \
  --threshold 3 \
  --shares 5 \
  --encoding mnemo-words \
  --in secret.bin \
  --out shares.txt
```

وزّع كل حصة من `shares.txt` إلى حاملها أو موقعها المحدد. لا تحتفظ بـ `shares.txt` كأثر طويل الأجل إلا إذا كان محمياً مثل السر الأصلي.

## استرداد الحصص المختارة

يجب أن يستخدم الاسترداد الحصص التي جُمعت للحدث المعتمد فقط.

```bash
safeparts combine \
  --in selected-shares.txt \
  --out recovered-secret.bin
```

عند الحاجة، ثبّت الترميز:

```bash
safeparts combine \
  --encoding mnemo-words \
  --in selected-shares.txt \
  --out recovered-secret.bin
```

## نمط ملف عبارة المرور

فضّل ملف عبارة المرور على كتابتها كوسيط أمر.

```bash
safeparts split \
  -k 3 \
  -n 5 \
  -e base64url \
  --passphrase-file passphrase.txt \
  --in secret.bin \
  --out shares.txt
```

يستخدم الاسترداد مصدر عبارة المرور نفسه:

```bash
safeparts combine \
  --passphrase-file passphrase.txt \
  --in selected-shares.txt \
  --out recovered-secret.bin
```

خزّن عبارة المرور عبر مسار سياسة منفصل. لا تضعها بجانب عدد كافٍ من الحصص للاسترداد.

## قالب runbook

<Steps>
1. اذكر اسم السر، والمالك، والعتبة، والترميز، وسياسة عبارة المرور.
2. حدد من يوافق على التقسيم والاسترداد.
3. حدد مجلد العمل وأوامر التنظيف.
4. قسّم سراً اصطناعياً بالأوامر نفسها.
5. قسّم السر الحقيقي.
6. تحقق من الاسترداد إذا سمحت السياسة بذلك.
7. وزّع الحصص وسجّل الحيازة دون تسجيل نص الحصص.
8. احذف الملفات المؤقتة التي تحتوي نصاً صريحاً أو كل الحصص أو عبارات المرور أو الناتج المستعاد.
9. جدولة تدريب ومراجعة حيازة.
</Steps>

## ملفات لا ينبغي الاحتفاظ بها

تجنّب الاحتفاظ بـ:

- ملف النص الصريح الأصلي؛
- ملف يحتوي كل الحصص؛
- ملفات عبارات المرور خارج المسار المعتمد؛
- ملفات السر المستعاد بعد استيراد السر أو تدويره؛
- transcripts الطرفية أو لقطات الشاشة التي تحتوي مادة حساسة.

للتدقيق، سجّل الموافقات والمشاركين والطوابع الزمنية وإصدارات الأوامر وحالة التدوير.
