يساعدك Safeparts على تجنب النسخة الاحتياطية الوحيدة والهشة. في خطة 2 من 3، مثلاً، تنشئ ثلاث حصص استرداد وتحتاج إلى أي حصتين منها لاستعادة السر.

استخدم هذه الوثائق لاختيار خطة استرداد، واختيار الواجهة المناسبة، وتجنب أخطاء التعامل الشائعة.

## ماذا يفعل Safeparts

- يقسّم بايتات عشوائية باستخدام مشاركة عتبة بأسلوب Shamir فوق <span class="token token--mono" dir="ltr">GF(256)</span>.
- يضيف وسم سلامة BLAKE3 حتى يفشل الاسترداد عند تلف الحصص أو خلطها.
- يمكنه التشفير قبل التقسيم باستخدام Argon2id وChaCha20-Poly1305 عند إضافة عبارة مرور.
- يوفّر نفس النواة عبر CLI وTUI وWASM والويب وسطح المكتب وواجهات Rust.

Safeparts ليس vault ولا مدير كلمات مرور ولا أداة seed لمحافظ العملات. خطة التخزين الخاصة بك هي حد الأمان الحقيقي.
