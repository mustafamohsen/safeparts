واجهة الويب موجودة في الصفحة الرئيسية (<span class="token token--mono" dir="ltr">/</span>).
عمليات التقسيم/الاسترداد تتم بالكامل داخل المتصفح عبر WASM.

<LinkButton href="/" icon="external" target="_blank" rel="noopener noreferrer">
	افتح تطبيق الويب
</LinkButton>

## التقسيم

<Steps>
1. الصق السر في **Secret**.
2. اختر <var>k</var> و <var>n</var>.
3. اختر الترميز.
4. (اختياري) أدخل عبارة مرور.
5. اضغط **Split** ثم خزّن كل حصة بشكل منفصل.
</Steps>

## الاسترداد

<Steps>
1. اختر الترميز (أو الصق حصة واترك الواجهة تكتشفه تلقائياً).
2. الصق على الأقل <var>k</var> حصص.
3. أدخل عبارة المرور إذا كنت استخدمتها.
4. اضغط **Combine**.
</Steps>

## حدود الخصوصية

التطبيق لا يرفع السر افتراضياً.
أي شيء تنسخه/تلصقه أو تخزنه هو مسؤوليتك.
