واجهة الويب موجودة في الصفحة الرئيسية (`/`). تستخدم WASM بحيث تتم عمليات التقسيم/الاستعادة
محلياً داخل المتصفح.

<LinkButton href="/" icon="external">افتح تطبيق الويب</LinkButton>

:::note
لا يتم رفع أي بيانات تلقائياً. البيانات لا تغادر جهازك إلا عند النسخ/اللصق أو حفظ الملفات
أو عند نشر نسخة معدّلة.
:::

## تقسيم سر

<Steps>
  1. أدخل السر في مربع **Secret**.
  2. اختر `k` (الحد الأدنى للحصص اللازمة للاستعادة) و `n` (عدد الحصص الكلي).
  3. اختر صيغة الحصة.
  4. (اختياري) عيّن عبارة مرور (تشفير قبل التقسيم).
  5. اضغط **Split** ثم خزّن كل حصة في مكان مختلف.
</Steps>

## استعادة سر من الحصص

<Steps>
  1. اختر صيغة الحصة (أو الصق حصة واترك الواجهة تكتشفها تلقائياً).
  2. الصق على الأقل `k` حصص.
  3. أدخل عبارة المرور إذا كانت الحصص مشفّرة.
  4. اضغط **Combine**.
</Steps>

:::caution
تجنّب استعادة الأسرار على أجهزة مشتركة، وتجنّب نسخ السر المستعاد إلى أدوات تقوم بالمزامنة
تلقائياً.
:::
