يستخدم Windows 11 تطبيق WinUI الأصلي، ويستخدم macOS 14 أو أحدث تطبيق SwiftUI الأصلي، بينما يستخدم Linux تطبيق Tauri. تعمل جميع تطبيقات سطح المكتب محلياً من دون خادم Safeparts أو telemetry أو خدمة خارجية أثناء التشغيل.

## متى تستخدمه

استخدم تطبيق سطح المكتب عندما تريد:

- مساحة عمل محلية مثبتة للتقسيم والاسترداد؛
- استيراد الملفات الثنائية واستردادها بدقة في Windows وmacOS؛
- استخدام ترميزات الحصص الأربعة والاسترداد التلقائي في Windows وmacOS؛
- تشغيل محلي دون نشر متصفح.

استخدم CLI أو TUI عندما تحتاج إلى سير عمل ثنائي موجه للملفات، أو كل الترميزات الأربعة، أو عمل كامل من الطرفية.

## نطاق تطبيق Tauri في Linux

تعرض واجهة Tauri في Linux الصيغ الأساسية نفسها الموجودة في تطبيق الويب:

- <span class="token" dir="ltr">base64url</span> وتظهر باسم <span class="token">Letters</span>؛
- <span class="token" dir="ltr">mnemo-words</span> وتظهر باسم <span class="token">Words</span>.

تدعم النواة الداخلية <span class="token" dir="ltr">base58check</span> و<span class="token" dir="ltr">mnemo-bip39</span>، لكنهما غير معروضين في واجهة Tauri على Linux حالياً. يعرض تطبيق Windows الأصلي الترميزات الأربعة وسير عمل الملفات الدقيق والاسترداد التلقائي.

## التقسيم

<Steps>
1. افتح التطبيق واختر <span class="token">تقسيم</span>.
2. أدخل السر.
3. اختر عدد الحصص المطلوبة وعدد الحصص الإجمالي وصيغة الحصة.
4. أضف عبارة مرور فقط إذا كانت خطة الاسترداد تخزنها بشكل منفصل.
5. قسّم وانسخ كل حصة إلى حاملها أو موقعها المحدد.
</Steps>

## الاسترداد

<Steps>
1. اختر <span class="token">استعادة</span>.
2. الصق الحصص المطلوبة.
3. أدخل عبارة المرور إذا استخدم التقسيم واحدة.
4. استعد السر وانقله فقط إلى وجهته المقصودة.
</Steps>

## البناء محلياً

```bash
cd desktop
bun install
bun run tauri:dev
```

لبناء واجهة إنتاجية:

```bash
bun run build
```

لبناء حزمة native على المنصة الحالية:

```bash
bun run tauri:build
```

راجع [التثبيت والاستضافة الذاتية](../build-and-run/) لحزم الإصدارات وأوامر البناء العامة.
