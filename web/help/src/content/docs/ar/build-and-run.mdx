## ثنائيات جاهزة (موصى بها)

حمّل آخر إصدار من GitHub:

- https://github.com/mustafamohsen/safeparts/releases

كل إصدار يتضمن:

- <span class="token token--mono" dir="ltr">safeparts</span> (CLI)
- <span class="token token--mono" dir="ltr">safeparts-tui</span> (واجهة طرفية)

تحقق سريع:

<Steps>
1. ضع الملفات التنفيذية على <span class="token token--mono" dir="ltr">PATH</span>.
2. شغّل <span class="token token--mono" dir="ltr">safeparts --help</span>.
3. شغّل <span class="token token--mono" dir="ltr">safeparts-tui --help</span>.
</Steps>

### Gatekeeper في macOS

إذا حمّلت الملفات التنفيذية من المتصفح، قد يضع macOS عليها علامة quarantine ويظهر التحذير "لا يمكن فتحه لأن المطور غير موثّق".

إذا بنيت من المصدر، غالباً لن ترى هذا التحذير.

إذا كنت تثق بالمصدر وتريد إزالة علامة quarantine:

```bash
# عدّل هذا المسار إلى مجلد الإصدار بعد فك الضغط
xattr -dr com.apple.quarantine /path/to/safeparts-release/

# أو أزلها لكل ملف تنفيذي
xattr -d com.apple.quarantine /path/to/safeparts
xattr -d com.apple.quarantine /path/to/safeparts-tui
```

## بناء من المصدر (Rust)

المتطلبات:

- Rust stable + <span class="token token--mono" dir="ltr">rustfmt</span> و <span class="token token--mono" dir="ltr">clippy</span>

البناء والاختبارات:

```bash
cargo build
cargo test --all-features
```

فحص بنمط CI:

```bash
cargo fmt --all -- --check
cargo clippy --all-targets --all-features -- -D warnings
```

## تطبيق الويب (Vite + React + WASM)

تطبيق الويب موجود في <span class="token token--mono" dir="ltr">web/</span>.
عمليات التقسيم/الاسترداد تتم داخل المتصفح عبر WASM.

المتطلبات:

- Bun
- <span class="token token--mono" dir="ltr">wasm-pack</span> (أو اترك سكربت البناء يثبته)

التشغيل محلياً:

```bash
cd web
bun install
bun run build:wasm
bun run dev
```

افتح http://localhost:5173.

## موقع المساعدة (Astro + Starlight)

الموقع موجود في <span class="token token--mono" dir="ltr">web/help/</span> ويُخدم تحت <span class="token token--mono" dir="ltr">/help/</span>.

```bash
cd web/help
bun install
bun run dev
```

افتح [http://localhost:4321/help/](http://localhost:4321/help/).
