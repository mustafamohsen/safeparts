---
title: Encodings
description: Choose a share text format for storage, copy and paste, or manual transcription.
---

Encodings change how a share packet is written down. They do not change how many shares are required, the secret sharing math, or passphrase protection.

Choose the format your recovery process can store and re-enter accurately.

## Formats

| Encoding | Best fit | Notes |
| --- | --- | --- |
| <span class="token">base64url</span> | Password managers, files, scripts, APIs | Compact URL-safe text. CLI alias: <span class="token">base64</span>. |
| <span class="token">base58check</span> | Manual copy and paste with compact text | Avoids visually ambiguous characters and includes a Base58Check checksum. CLI alias: <span class="token">base58</span>. |
| <span class="token">mnemo-words</span> | Paper, metal, and spoken recovery | Uses word groups with CRC16 to catch many transcription errors. |
| <span class="token">mnemo-bip39</span> | Workflows that expect BIP-39-valid word sequences | Safeparts share text can look like wallet words, but it is not a wallet seed. |

## Availability by interface

| Interface | Split output | Combine input |
| --- | --- | --- |
| Web app | <span class="token">base64url</span>, <span class="token">mnemo-words</span> | Same two formats, with best-effort detection on paste. |
| Desktop app | <span class="token">base64url</span>, <span class="token">mnemo-words</span> | Same two formats, with local Tauri commands. |
| CLI | All four concrete encodings | Auto-detect when `--encoding` is omitted, or pass a concrete encoding. |
| TUI | All four concrete encodings | Auto-detect or choose a concrete encoding. |
| Rust library | All four concrete encodings through `safeparts_core::encoding` | Auto parse and concrete parse APIs. |

## Choosing a format

Use <span class="token">base64url</span> when machines will store or move the shares. It is the shortest common option and works well in files and vault fields.

Use <span class="token">base58check</span> when operators may read compact text by eye and you want an encoding-level checksum.

Use <span class="token">mnemo-words</span> when humans will copy, print, read, or engrave shares.

Use <span class="token">mnemo-bip39</span> only when BIP-39-valid word sequences help your storage process.

Mnemonic Safeparts shares are not wallet seed phrases. Do not import them into wallet software.
