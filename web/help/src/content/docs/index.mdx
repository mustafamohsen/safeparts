Safeparts helps you avoid one fragile backup. It turns one secret into <var>n</var> recovery shares and requires any <var>k</var> of them to recover the original value.

Use these docs to choose a recovery plan, pick an interface, and avoid common handling mistakes.

## What Safeparts does

- Splits arbitrary bytes with Shamir-style threshold sharing over <span class="token token--mono">GF(256)</span>.
- Adds a BLAKE3 integrity tag so recovery fails when shares are corrupted or mixed.
- Can encrypt before splitting with Argon2id and ChaCha20-Poly1305 when you add a passphrase.
- Exposes the same core through CLI, TUI, WASM, web, desktop, and Rust APIs.

Safeparts is not a vault, password manager, or wallet seed tool. Your storage plan is still the security boundary.
