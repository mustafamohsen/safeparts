Use `safeparts_core` when a Rust application needs Safeparts behavior without shelling out to the CLI.

The crate works on bytes. It handles split, combine, packet validation, integrity checks, text encodings, and optional passphrase protection. Your application still owns storage, authorization, audit policy, and secret lifetime.

:::caution
Do not log secrets, recovered bytes, passphrases, encoded shares, or packet payloads.
:::

## Add the crate

From another repository, use a Git dependency until a published crate is available:

```toml
[dependencies]
safeparts_core = { git = "https://github.com/mustafamohsen/safeparts", package = "safeparts_core" }
```

Inside this workspace, depend on the workspace crate according to the local crate layout.

If your application wraps sensitive buffers, add `zeroize`:

```toml
[dependencies]
zeroize = "1"
```

## Split and combine bytes

```rust
use safeparts_core::{combine_shares, split_secret, CoreResult};

fn split_then_recover() -> CoreResult<()> {
    let secret = b"synthetic integration secret";

    let shares = split_secret(secret, 2, 3, None)?;
    let recovered = combine_shares(&shares[..2], None)?;

    assert_eq!(recovered, secret);
    Ok(())
}
```

The core threshold range is <span class="token" dir="ltr">1 &lt;= k &lt;= n &lt;= 255</span>. Recovery needs at least <var>k</var> packets from the same split set.

## Encode shares for storage

```rust
use safeparts_core::encoding::{self, Encoding};
use safeparts_core::{split_secret, CoreResult};

fn split_for_storage() -> CoreResult<Vec<String>> {
    let packets = split_secret(b"synthetic storage secret", 2, 3, None)?;

    packets
        .iter()
        .map(|packet| encoding::encode_packet(packet, Encoding::Base64url))
        .collect()
}
```

Store each encoded share in a separate failure domain. Keeping enough shares together defeats the threshold plan.

## Parse and recover pasted shares

```rust
use safeparts_core::encoding::{self, Encoding};
use safeparts_core::{combine_shares, CoreResult};

fn recover_from_paste(input: &str) -> CoreResult<Vec<u8>> {
    let parsed = encoding::parse_share_packets(input, Encoding::Auto)?;
    combine_shares(&parsed.packets, None)
}
```

Use `parse_share_packets_wrapped_mnemonics` for text areas where one mnemonic share may wrap across multiple lines. Blank lines separate wrapped mnemonic shares.

## Add passphrase protection

```rust
use safeparts_core::{combine_shares, split_secret, CoreResult};
use zeroize::Zeroizing;

fn encrypted_round_trip() -> CoreResult<()> {
    let passphrase = Zeroizing::new(b"synthetic passphrase for tests".to_vec());

    let shares = split_secret(
        b"synthetic protected secret",
        2,
        3,
        Some(passphrase.as_slice()),
    )?;

    let recovered = combine_shares(&shares[..2], Some(passphrase.as_slice()))?;
    assert_eq!(recovered, b"synthetic protected secret");
    Ok(())
}
```

Packets store the salt, nonce, and Argon2id parameters needed for recovery. They do not store the passphrase.

## Public API reference

### Top-level crate API

| Item | Use |
| --- | --- |
| `split_secret(secret, k, n, passphrase)` | Main API for creating share packets. |
| `combine_shares(packets, passphrase)` | Main API for recovering secret bytes. |
| `tag_and_split(secret, k, n)` | Compatibility wrapper for unprotected splits. Prefer `split_secret` in new code. |
| `combine_and_verify(packets)` | Compatibility wrapper for unprotected combine. Prefer `combine_shares` in new code. |
| `CoreError` | Typed error enum. |
| `CoreResult<T>` | Result alias for core operations. |

### `encoding` module

| Item | Use |
| --- | --- |
| `Encoding` | `Auto`, `Base64url`, `Base58check`, `MnemoWords`, and `MnemoBip39`. |
| `Encoding::CONCRETE` | Concrete output encodings. |
| `Encoding::WITH_AUTO` | Concrete encodings plus auto parsing. |
| `Encoding::label()` | Canonical label for UI or storage metadata. |
| `Encoding::parse_name(name)` | Parses canonical names and CLI aliases. |
| `ParsedSharePackets` | Parsed packets plus detected encoding. |
| `encode_packet(packet, encoding)` | Encode one packet as text. |
| `decode_packet(text, encoding)` | Decode one packet. |
| `parse_share_packets(input, encoding)` | Parse one or more shares. |
| `parse_share_packets_wrapped_mnemonics(input, encoding)` | Parse text area input with wrapped mnemonic shares. |
| `detect_encoding(input)` | Detect likely encoding without returning packets. |

### `packet` module

| Item | Use |
| --- | --- |
| `SharePacket` | Packet with `set_id`, `k`, `n`, `x`, payload, and optional crypto params. |
| `SharePacket::is_encrypted()` | Returns true when passphrase protection was used. |
| `SharePacket::encode_binary()` | Serialize to Safeparts binary packet format. |
| `SharePacket::decode_binary(bytes)` | Parse the binary packet format. |
| `packet::binary_total_len(bytes)` | Compute full packet length from a byte prefix. |

## Application responsibilities

`safeparts_core` does not decide:

- where each share is stored;
- who may provide a share;
- whether passphrases are required;
- how recovered bytes move through memory or disk;
- which audit events are safe to record.

A safe audit event says that recovery was attempted or completed under an approved policy. It should not include share text or recovered values.
