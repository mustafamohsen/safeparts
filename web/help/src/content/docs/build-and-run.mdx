## Prebuilt binaries (recommended)

Download the latest release from GitHub:

- https://github.com/mustafamohsen/safeparts/releases

Each release includes:

- <span class="token token--mono">safeparts</span> (CLI)
- <span class="token token--mono">safeparts-tui</span> (terminal UI)

Quick sanity check:

<Steps>
1. Put the binaries on your <span class="token token--mono">PATH</span>.
2. Run <span class="token token--mono">safeparts --help</span>.
3. Run <span class="token token--mono">safeparts-tui --help</span>.
</Steps>

### macOS Gatekeeper

If you download the release binaries via a browser, macOS may quarantine them and show "cannot be opened because the developer cannot be verified".

If you built from source, you usually will not see this.

If you trust the download and want to remove the quarantine attribute:

```bash
# Point this at the folder you extracted the release into
xattr -dr com.apple.quarantine /path/to/safeparts-release/

# Or remove it per-binary
xattr -d com.apple.quarantine /path/to/safeparts
xattr -d com.apple.quarantine /path/to/safeparts-tui
```

## Build from source (Rust)

Prereqs:

- Rust stable + <span class="token token--mono">rustfmt</span> and <span class="token token--mono">clippy</span>

Build + tests:

```bash
cargo build
cargo test --all-features
```

CI-style lint:

```bash
cargo fmt --all -- --check
cargo clippy --all-targets --all-features -- -D warnings
```

## Web UI (Vite + React + WASM)

The web app lives in <span class="token token--mono">web/</span>.
It runs split/combine in your browser via WASM.

Prereqs:

- Bun
- <span class="token token--mono">wasm-pack</span> (or let the build script install it)

Run locally:

```bash
cd web
bun install
bun run build:wasm
bun run dev
```

Open http://localhost:5173.

## Help/docs site (Astro + Starlight)

This site lives in <span class="token token--mono">web/help/</span> and is served under <span class="token token--mono">/help/</span>.

```bash
cd web/help
bun install
bun run dev
```

Open [http://localhost:4321/help/](http://localhost:4321/help/).
