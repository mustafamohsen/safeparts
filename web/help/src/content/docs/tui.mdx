---
title: TUI
description: Interactive terminal UI for hands-on splitting and recovery.
---

The TUI binary is `safeparts-tui` (crate: `crates/safeparts_tui`).

Run directly:

```bash
cargo run -q -p safeparts_tui
```

Or via the CLI launcher:

```bash
cargo run -q -p safeparts -- tui
```

## Key shortcuts

- <kbd>Enter</kbd>: run split/combine
- <kbd>Ctrl</kbd>+<kbd>L</kbd>: load secret/share file(s)
- <kbd>Ctrl</kbd>+<kbd>S</kbd>: save (one file per share, or recovered secret)
- <kbd>Ctrl</kbd>+<kbd>C</kbd>: copy (UTF-8 if possible, else <span class="token">base64</span>)
- <kbd>Ctrl</kbd>+<kbd>Q</kbd>: quit
- <kbd>?</kbd>: help overlay

:::note
Clipboard support uses your system clipboard when available, and falls back to
OSC52 escape sequences for terminals that support it (useful over SSH/tmux).
:::
