The TUI binary is <span class="token token--mono">safeparts-tui</span>. You can also start it through the CLI with <span class="token token--mono">safeparts tui</span>.

Use the TUI when you want a guided terminal workflow, offline operation, clipboard support, and file load or save actions.

## What the TUI supports

- Split and combine workflows.
- All concrete encodings: <span class="token">base64url</span>, <span class="token">base58check</span>, <span class="token">mnemo-words</span>, and <span class="token">mnemo-bip39</span>.
- Auto-detection on combine.
- Optional passphrases.
- Loading secrets or share files.
- Exporting one file per share or saving recovered bytes.

## Offline workflow

<Steps>
1. Use a machine with the network disconnected if your procedure requires it.
2. Run <span class="token token--mono">safeparts-tui</span>.
3. Choose <var>k</var>, <var>n</var>, and an encoding.
4. Enter or load the secret.
5. Split and export shares to separate media or holders.
6. Recover once with <var>k</var> shares if your policy allows verification.
</Steps>

:::caution
Offline execution only helps if storage is separated. Do not put enough shares to recover on the same USB drive, account, or safe.
:::

## Keyboard shortcuts

| Keys | Action |
| --- | --- |
| <kbd>Tab</kbd> / <kbd>Shift</kbd>+<kbd>Tab</kbd> | Move focus. |
| <kbd>Left</kbd> / <kbd>Right</kbd> | Switch between split and combine tabs. |
| <kbd>Up</kbd> / <kbd>Down</kbd> | Change numeric values, selected encoding, or selected share depending on focus. |
| <kbd>Enter</kbd> | Run split or combine. |
| <kbd>Ctrl</kbd>+<kbd>L</kbd> | Load a secret file or share files. |
| <kbd>Ctrl</kbd>+<kbd>S</kbd> | Save or export output. |
| <kbd>Ctrl</kbd>+<kbd>C</kbd> | Copy selected share, all shares, or recovered output depending on focus. |
| <kbd>Ctrl</kbd>+<kbd>V</kbd> | Paste into the focused editor when clipboard access is available. |
| <kbd>Ctrl</kbd>+<kbd>U</kbd> | Clear the focused passphrase field. |
| <kbd>?</kbd> or <kbd>F1</kbd> | Open help. |
| <kbd>Esc</kbd> | Close help or cancel a modal. |
| <kbd>Ctrl</kbd>+<kbd>Q</kbd> | Quit. |
